## Repository for BIOC 6242 course materials, Fall 2022

Suggested Terminal applications, for major operating systems:

Windows:
- [Git for Windows](https://git-scm.com/download/win)
- [Cygwin](https://www.cygwin.com/install.html)
- [WSL2 activation](https://docs.microsoft.com/en-us/windows/wsl/install) (Advanced! - sets up (windowless) Linux virtual machines under the Hyper-V hypervisor)
- [VirtualBox](https://www.virtualbox.org/wiki/Downloads) (Advanced! - sets up full X-windowed virtual machines under the VirtualBox hypervisor)

**NB**: WSL and VirtualBox don't work well together. Avoid installing both unless you are _very_ comfortable working with virtual machines!

Mac OSX/*nix:
- Built-in terminal, or custom terminals for advanced users
